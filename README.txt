Автономный статический тренажёр экспозиции

Публичные адреса:
- RU: https://green4.photo/exposure-triangle-trainer/
- UA: https://green4.photo/exposure-triangle-trainer/uk/

Локальный просмотр:
1. Откройте index.html двойным щелчком — RU-версия работает через file://.
2. Переключатель UA открывает uk/index.html; обе страницы используют общую
   папку assets через относительные пути.

Размещение на сервере:
1. Загрузите папку exposure-triangle-trainer из установочного архива в корень сайта.
2. Внутри неё должны находиться index.html, assets и uk/index.html.
3. Относительные пути сохраняют работоспособность всей папки при переносе.
4. Canonical и метаданные соцсетей указывают на согласованные публичные URL.

Для работы не требуются PHP, база данных, Node.js или система сборки.

Основные файлы:
- assets/css/style.css — общие стили RU и UA;
- assets/js/trainer.js — общая логика и локализованные тексты;
- assets/images/logo.png — логотип, favicon и изображение для соцсетей.

Возможности:
- свободный калькулятор экспозиции;
- режим тренировки со случайными заданиями, проверкой и подсказкой;
- счётчик решённых заданий и серия правильных ответов;
- чистый калькулятор после каждой перезагрузки страницы;
- сохранение в браузере только статистики тренировки;
- ссылка на текущую комбинацию и перенос её при смене языка.

Проверка:
php tests/validate.php

Архив exposure-triangle-trainer-redesign.zip — зафиксированная версия после
технического редизайна, до добавления нового режима тренировки.
